YouTube Video Q&A Chatbot – Generative AI Conversational Agent
Nov 2024 – Apr 2025
• Developed a Streamlit web app that enables users to ask natural language questions about any YouTube video by leveraging its
transcript, OpenAI LLMs, and Retrieval Augmented Generation (RAG)
• Automated extraction and semantic chunking of YouTube video captions, followed by embedding with OpenAI models and indexing
using a FAISS vector store for efficient context retrieval.
• Engineered a conversational pipeline that retrieves relevant transcript segments and dynamically generates answers via
LLM prompt templates, supporting flexible, context-aware video Q&A.
• Deployed with a secure, user-friendly interface for seamless OpenAI API integration and session-based interaction.
